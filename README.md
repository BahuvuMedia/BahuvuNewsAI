\# BahuvuNewsAI



BahuvuNewsAI is an AI-powered Telugu news video generator.



\## Features



\* Fetches the latest news from Google News RSS

\* Filters unwanted news

\* Downloads a news image

\* Generates a Telugu news script using AI

\* Creates Telugu voice narration

\* Produces an MP4 news video automatically



\## Project Structure

\## Installation



```bash

pip install -r requirements.txt

```



\## Run



```bash

python main.py

```



\## Output



The generated files are saved in the `outputs` folder.
